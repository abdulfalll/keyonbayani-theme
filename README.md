# KeyonBayani Child Theme

A custom WordPress child theme for **Divi**, built for the KeyonBayani e-commerce and portfolio platform.

## Features
- **Custom Assets:** Enqueued custom CSS and JS architecture.
- **WooCommerce Ready:** Optimized for custom e-commerce template overrides.
- **Clean Enqueueing:** Properly handles parent Divi styles to ensure fast loading.

## Tech Stack
- PHP / WordPress
- Divi Parent Theme
- CSS3 / JavaScript
